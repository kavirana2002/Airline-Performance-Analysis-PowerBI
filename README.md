# ✈️ GLOBAL AIRLINE PERFORMANCE TRACKER DASHBOARD | POWER BI

A beginner-friendly Power BI dashboard project that visualizes and analyzes airline performance data. This project was built as a personal learning exercise to explore data visualization, KPI tracking, and interactive reporting using Microsoft Power BI.

---

## 📌 Project Overview

This dashboard provides a comprehensive view of airline performance metrics, helping users understand flight patterns, on-time performance, route efficiency, and operational trends through interactive visuals.

---

## 🎯 Objective

To analyze and visualize airline performance data using Power BI, and gain hands-on experience with data modeling, DAX measures, and dashboard design.

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| **Microsoft Power BI** | Data visualization & dashboard creation |
| **Power Query** | Data cleaning & transformation |
| **DAX** | Calculated measures & KPIs |
| **JSON** | Data source |

---

## 📊 Dashboard Features

- 📈 **Charts & Graphs** — Bar charts, line graphs, and pie charts showing trends in flight delays, cancellations, and performance over time
- 🔢 **KPIs & Cards** — Key metrics like total flights, on-time rate, average delay, and cancellation rate displayed as quick-glance cards
- 📋 **Tables & Filters** — Detailed data tables with slicers to filter by airline, route, date range, or airport

## 🚀 How to Use

1. **Clone or download** this repository
2. Open the `.pbix` file in **Microsoft Power BI Desktop**
   - Download Power BI Desktop for free from [here](https://powerbi.microsoft.com/desktop/)
3. If needed, update the **data source path** to point to the dataset on your machine
4. Explore the dashboard using the filters and slicers!
